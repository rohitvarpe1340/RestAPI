A simple and scalable REST API built using Node.js, Express, and MySQL with ES6 modules and dotenv configuration.
Features
* Create User (POST)
* Get All Users (GET)
* Get User by ID (GET)
* Update User (PUT & PATCH)
* Delete User (DELETE)
* MySQL Database Integration
* Environment Variables using dotenv
* Clean and structured code

<<<----- Setup Instructions------>>>
1) Install dependencies
npm install
2) Setup environment variables (.env)
PORT=4000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=testdb
   3) Run the server
  npm run dev
